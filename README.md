# Superstore
This project consists of a front-end written in React.js, a back-end written in C#, and MS SQL stored procedures.

## Superstore API

### Address
- GET /api/addresses
- GET /api/addresses/{id}
- POST /api/addresses
- PUT /api/addresses/{id}
- DELETE /api/addresses/{id}
